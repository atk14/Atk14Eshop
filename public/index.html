File icons

Flexible utility for creating file icons. ###Very basic file icon ###File icon with image Add modifier class with file type extension like fileicon-zip. Many common file types have special icon, if not, default icon is displayed. Only few of many available file types are displayed here. For best results, use larger font sizes.

###File icon with color Add fileicon-color class and many common file type icons will be colored with its typical color. If color for given file type is not found, default dark gray color is used.

###File icons with text If you want to have text instead of image on icon, just add data-icon-text attribute with desired text. Text should have no more than 3 or 4 characters. For best results font size should be even larger.

###Note for developers about PhotoSwipe Basic markup necessary to build zoomable Photoswipe gallery or single image:
Gallery is wraped .gallery__images or .iobject--picture class. If other class is used, initPhotoSwipeFromDOM parameters in application.js must be adjusted. Each image is wraped in figure element which includes link, thumbnail image and optional figcaption. Link tag contains some additional attributes: - data-size (required): widthxheight size of large image - data-minithumb (optional): URL of very small (such as 32 px) thumbnail used for zooming animation. It has the same aspect ratio as large image. Recommended in cases when normal thumbnail has different aspect ratio than large image.
